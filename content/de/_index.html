Direkt zur Webapp
{{< /blocks/cover >}} {{% blocks/lead color="primary" %}} Eine Webanwendung, die es ermöglicht Teilnehmer, Wettbewerbe, Gruppen und Wertungen eines Poetry Slam zu verwalten und während der Veranstaltung auf einem Beamer zu präsentieren. {{% /blocks/lead %}} {{< blocks/section color="dark" >}} {{% blocks/feature icon="fa-lightbulb" title="Keine Installation" %}} Die Webapp funktioniert auf allen Betriebssystemen. Notwendig ist nur ein Google Chrome {{% /blocks/feature %}} {{% blocks/feature icon="fab fa-github" title="Open Source und kostenlos" url="https://github.com/opendata-heilbronn/poetryslam" %}} Die Software wird Open Source entwickelt. Wer mithelfen will ist gerne eingeladen. Fehlermeldungen und Wünsche bitte als Issue. {{% /blocks/feature %}} {{% blocks/feature icon="fab fa-twitter" title="Folge und auf Twitter!" url="https://twitter.com/codeforhn/" %}} Für mehr Projekte und Informationen rund um Code for Heilbronn e.V. {{% /blocks/feature %}} {{< /blocks/section >}} {{< blocks/section >}}
